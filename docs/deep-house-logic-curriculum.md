# Deep House Logic Curriculum

This is a practical 10-week training plan for getting better at producing house music in Logic with a deep, groove-forward center. It is designed for 3-5 hours per week, using one long session and one short session, and it assumes you already bring usable rhythm and keyboard instincts from drums and piano.

The goal is not to become an expert in every production topic. The goal is to make strong demos quickly, turn loops into arranged ideas, and finish one track that is clearly stronger than what you could make at the start.

## Ground Rules

- Stay inside Logic and a small tool palette for the first block.
- Use references every week.
- Build sections early instead of polishing 8-bar loops forever.
- Favor groove, restraint, and movement over adding more layers.
- Finish the session with a bounce and written next steps.

## Core Outcome For This Block

By the end of week 10, you should be able to:

- open Logic and start a credible house idea quickly
- write drums, bass, and chords that lock together
- turn a loop into a 2-4 minute arranged demo
- compare your work objectively against references
- complete one strongest full track

## Reference Framework

Start with three references that cover different strengths of the lane you want to enter. Do not treat them as songs to copy exactly. Treat them as calibration tools.

### Reference 1: `ANOTR - Relax My Eyes`

Use this for groove economy and arrangement restraint.

Targets:

- Minimal but confident drum programming
- Repetition with subtle movement instead of constant new sections
- Strong pocket from percussion placement and dynamics
- A track that feels expensive without sounding busy

What to listen for:

- How often new elements actually enter
- Where swing and loose feel appear
- How the groove survives even when the arrangement thins out
- How transitions are often handled with subtraction, filtering, and texture

What to practice from it:

- Build a loop using only kick, clap, hats, percussion, bass, one chord part, and one ear-candy element
- Make three variations of the same 16-bar section without adding a new main instrument

### Reference 2: `Black Coffee - Drive`

Use this for low-end patience, spaciousness, and emotional control.

Targets:

- Bass that supports the record instead of dominating it
- Chords that feel musical but not overplayed
- Space between elements
- Slow-burn development with intentional tension and release

What to listen for:

- How the bass leaves room for kick and vocals
- How harmonic parts repeat with small articulation changes
- How atmosphere creates depth without clutter
- How long the track can sit on one strong idea before moving

What to practice from it:

- Write an 8-bar bass pattern that uses fewer notes than you think you need
- Pair it with a chord part that adds mood without stepping on the bass rhythm

### Reference 3: `Kerri Chandler - Rain`

Use this for house fundamentals, swing, and musical pocket.

Targets:

- House drums that feel human and alive
- Chords with soul and timing feel
- Bass and drums behaving like one rhythm section
- Call-and-response between groove elements

What to listen for:

- The relationship between kick, clap, hats, and chord stabs
- The timing feel of chords against straight grid drums
- The push-pull that creates movement even in simple loops
- The role of velocity and micro-timing in house feel

What to practice from it:

- Program a classic house drum loop, then shift percussion and chord timing slightly off-grid by ear
- Play chord stabs rhythmically instead of holding long pads

## How To Use References Each Week

Every long session starts with 10-15 minutes of active listening.

Use this sequence:

1. Pick one reference.
2. Write down the rough structure by time: intro, groove-in, lift, drop, breakdown, outro.
3. Note only three things:
   - what carries the groove
   - what the bass is doing
   - what changes every 8 or 16 bars
4. Do not analyze plugins, sound design tricks, or mixing secrets yet.
5. Apply one observation directly in your own project that day.

## Limited Tool Palette

For the first 10 weeks, restrict yourself as much as possible.

### Drums And Samples

- One main kick folder
- One clap or snare folder
- One hats and percussion folder
- A small set of loops or one-shots you already like

Rule:

- If you audition more than 10 kick samples in a session, stop and commit to the best two.

### Logic Instruments And Effects

Use mostly stock tools at first:

- `Drum Machine Designer` or `Quick Sampler`
- `Alchemy` for pads, plucks, and textures
- `Retro Synth` or `ES2` for basses and simple leads
- `Channel EQ`
- `Compressor`
- `AutoFilter`
- `Chromaverb`
- `Delay Designer` or `Stereo Delay`
- `Direction Mixer`
- `Limiter`

Rule:

- Learn what these do in context before chasing third-party plugin solutions.

## Session Rhythm

Your week should be predictable.

### Long Session: 2-3 Hours

Use this structure every week:

1. `15 min`: reference listening and note-taking
2. `20 min`: sound selection and template setup
3. `60-90 min`: core creation or arrangement work
4. `15 min`: bounce, compare, and write next steps

Default focus for the long session:

- Week 1-4: idea generation plus arrangement markers
- Week 5-7: converting ideas into demos
- Week 8-10: finishing the strongest track

### Short Session: 45-75 Minutes

Choose one lane only:

- groove drill
- bass/chord drill
- arrangement pass
- transition pass
- reference comparison pass

Rule:

- If the short session turns into random browsing or tinkering, stop and go back to a single drill.

## Default Logic Workflow

Use this order whenever you start a new idea:

1. Set tempo and rough vibe.
2. Build an 8 or 16 bar drum loop.
3. Add bass before adding complex chords.
4. Add one harmonic layer only.
5. Create section markers immediately: intro, A, B, break, return, outro.
6. Duplicate the loop across a rough arrangement.
7. Remove elements to create energy contrast.
8. Add transitions only after the sections exist.
9. Do a rough level balance.
10. Bounce and review.

This order matters. It protects you from getting stuck in beautiful but useless loops.

## Skill Sequence

Study and practice skills in this order.

### 1. Arrangement Habits

This is the highest-priority weakness to solve first.

Focus:

- making sections early
- changing energy every 8 or 16 bars
- learning how little you need for a section change
- ending each session with a structure, not just a loop

Do not:

- spend an hour EQing a loop with no arrangement

### 2. Groove Quality

Your drumming background should help here if you stay attentive to feel.

Focus:

- kick and bass relationship
- hi-hat placement
- percussion syncopation
- swing and micro-timing
- velocity differences that create movement

Do not:

- quantize every element to death

### 3. Bass And Harmony Lock

Your piano background gives you an advantage here, but house needs discipline.

Focus:

- fewer notes in the bass
- chord rhythms that complement the bass
- avoiding frequency and rhythmic collisions
- using repetition on purpose

Do not:

- write a busy bass and busy chords at the same time

### 4. Sound Selection

Good house often sounds better because the source choices are better, not because the mix is more complicated.

Focus:

- selecting sounds that already belong together
- avoiding too many overlapping midrange parts
- choosing one identity element per track

Do not:

- stack three average sounds when one good sound will do

### 5. Mixing Basics

Only go as deep as needed to make the idea translate.

Focus:

- rough levels
- EQ for space
- basic compression
- low-end clarity
- mono awareness for kick and bass

Do not:

- try to master the track during this block

## Accountability System

You said checklists, deadlines, and reference comparison help you most. Use all three.

### Weekly Deadline

Each week must end with one exported output:

- one loop
- one arranged demo
- one transition pass
- one revised mix pass
- or one full track bounce

No week ends with only notes or unfinished ideas.

### Session Checklist

Use this at the end of every session:

- Did I work from a reference first?
- Did I produce audio or MIDI instead of mainly browsing sounds?
- Did I improve groove, bass/chords, or arrangement in a measurable way?
- Did I bounce the result?
- Did I write the next three actions before closing Logic?

If three or more answers are `no`, the session was not focused enough.

### Demo Checklist

A demo counts as valid only if:

- it has clear sections, not just a loop
- the drums feel intentional
- the bass and harmonic layer do not fight each other
- one memorable sonic idea is present
- the arrangement changes at least every 8 or 16 bars
- the bounce sounds closer to a reference than last week's work

### Reference Comparison Rubric

Score each item from `1-5` against one reference:

- Groove: does the track move physically?
- Low end: are kick and bass readable?
- Restraint: are there too many parts?
- Energy curve: can you feel section changes?
- Identity: is there one recognizable character element?

Interpretation:

- `1-2`: weak, needs deliberate revision
- `3`: acceptable for draft stage
- `4`: strong for your current level
- `5`: reference-grade in that category

Do not score overall vibe. Score categories separately so the fix becomes obvious.

## 10-Week Curriculum

This is the working block. If life gets busy, stretch it to 12 weeks instead of compressing it.

### Week 1: Logic Startup Workflow

Primary goal:

- remove friction so opening Logic immediately leads to work

Long session:

- build a simple template
- load a small drum palette
- create one 16-bar groove
- add markers for intro, groove, break, return, outro

Short session:

- recreate one drum groove by ear from a reference

Output:

- one bounced 16-bar groove with markers in place

### Week 2: Kick, Bass, Pocket

Primary goal:

- make the rhythm section feel connected

Long session:

- write three bass variations over one drum loop
- keep note count low
- choose one version and sketch arrangement changes

Short session:

- mute everything except kick and bass and fix the pocket

Output:

- one 60-90 second draft centered on kick and bass

### Week 3: Chords Without Clutter

Primary goal:

- add harmony that supports the groove

Long session:

- write one chord part that leaves room for bass
- test stab rhythm versus sustained chords
- arrange two contrasting sections

Short session:

- voice the same chord progression three simpler ways

Output:

- one arranged sketch with drums, bass, and one chord layer

### Week 4: Arrangement Discipline

Primary goal:

- stop treating arrangement as an afterthought

Long session:

- duplicate your best idea into a 2-3 minute structure
- create contrast by muting, filtering, and dropping elements

Short session:

- add only transitions and section changes

Output:

- first full arranged demo, even if rough

### Week 5: Groove Revision Week

Primary goal:

- improve feel instead of adding parts

Long session:

- revisit demo 1 and improve hats, percussion, swing, and velocities
- compare directly with `ANOTR - Relax My Eyes`

Short session:

- 30-45 minutes of micro-timing and velocity edits by ear

Output:

- revised demo 1 with stronger groove

### Week 6: Second Demo From Scratch

Primary goal:

- prove the workflow works again

Long session:

- start a new idea from scratch using the default workflow
- get to a 90-second arranged draft in one sitting

Short session:

- identify which of demo 1 or demo 2 has better long-term potential

Output:

- second arranged demo

### Week 7: Bass And Harmony Focus

Primary goal:

- make the musical engine of the track more convincing

Long session:

- improve the weaker area in both demos: either bass rhythm or chord support
- commit to the stronger demo as the finishing candidate

Short session:

- analyze one reference only for bass/chord interplay

Output:

- one chosen demo promoted to track candidate

### Week 8: Finish Candidate Structure

Primary goal:

- turn the chosen demo into a full track structure

Long session:

- complete intro, main sections, break, return, and outro
- ensure each section has a job

Short session:

- fix dead sections or sections that overstay their welcome

Output:

- full-track arrangement bounce

### Week 9: Sonic Identity And Transitions

Primary goal:

- make the track feel intentional and memorable

Long session:

- strengthen one identity element
- refine transitions, ear candy, filters, and automation

Short session:

- compare against references and remove anything unnecessary

Output:

- near-final track bounce

### Week 10: Basic Mix And Release Decision

Primary goal:

- get the track to honest demo quality and close the block

Long session:

- set levels
- clean low end
- reduce mud
- make kick and bass readable
- export final demo

Short session:

- do one final reference comparison and write lessons learned

Output:

- one strongest finished track
- one short postmortem with what improved and what still needs work

## What To Do When You Get Stuck

If you are stuck:

- mute parts until the groove returns
- return to kick and bass
- listen to 60 seconds of a reference
- create markers and force a section change
- bounce the current version and review away from the screen

If you are still stuck after 15 minutes:

- stop building and switch to a drill

## Drills Library For Short Sessions

Use these when motivation is low or you only have limited time.

### Groove Drill

- program 8 bars with kick, clap, closed hat, open hat, one percussion element
- make it feel better using only timing and velocity

### Bass Drill

- write a one-bar bass motif
- create four variations using rhythm changes only

### Chord Drill

- play a progression with only triads
- then rewrite it with fewer notes and stronger rhythm

### Arrangement Drill

- take a 16-bar loop
- create intro, main, break, and return using subtraction only

### Reference Drill

- map the first 90 seconds of a reference by section and energy

## Simple Rules For Faster Progress

- Finish ugly arrangements before polishing pretty loops.
- Use fewer sounds than you want to.
- If the bass is busy, simplify the chords.
- If the arrangement is dead, remove elements before adding elements.
- If you cannot explain what a layer adds, mute it.
- Bounce often.

## End-Of-Block Review

At the end of the block, answer these questions:

- Can I start faster in Logic than before?
- Do my demos now reach arrangement stage reliably?
- Is my groove more confident and less stiff?
- Do my bass and chords cooperate better?
- Did I finish one track without endless restarting?

If most answers are `yes`, repeat the cycle with slightly higher standards and a fresh set of references.
