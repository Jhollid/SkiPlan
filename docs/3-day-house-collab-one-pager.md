# 3-Day House Collab One-Pager

Keep this open during the session. The goal is simple: have fun, capture good grooves fast, and leave with one track worth finishing.

## Core Rules

- Start with a groove, not a conversation.
- Record early.
- Keep parts minimal.
- If it feels good, capture it immediately.
- If stuck for 10-15 minutes, simplify or pivot.
- Bounce something before stopping.

## Roles

You:

- drums
- Logic control
- chords, textures, arrangement

Friend:

- bass writing
- live bass takes
- groove feedback

Shared:

- choose the best idea fast
- kill weak parts quickly

## Session Template

### 1. Warm-Up

`15-20 min`

- play one reference
- name one thing to borrow
- start jamming immediately

### 2. Main Block

`75-100 min`

- build one idea
- record bass early
- keep multiple takes
- add only one harmonic layer once groove works

### 3. Review Or Arrange

`20-30 min`

- stretch into sections
- or compare sketches and choose one

### 4. End Strong

`10 min`

- bounce
- label clearly
- write next move for tomorrow

## Day Targets

### Day 1

- make 2-3 groove sketches
- record several bass takes
- choose nothing yet unless one idea is obviously best

### Day 2

- pick the strongest sketch within 10 minutes
- build intro, main groove, breakdown, return, outro
- add one identity element only

### Day 3

- open the best track only
- tighten timing, transitions, and arrangement
- do a simple balance pass
- export final demo

## Quick Fixes

### If groove feels weak

- mute to drums and bass
- change bass rhythm before changing sounds

### If track feels busy

- remove one percussion layer
- mute one harmonic part

### If section feels flat

- subtract elements
- automate filter, delay, or reverb

### If energy in the room drops

- stop editing
- play and record another take

## Bass Recording Rules

- record DI clean
- keep full takes
- do not over-edit good feel
- save one extra take

## End-Of-Day Questions

- Which idea had the most pull?
- Which bass take felt best?
- What is the first move tomorrow?

## Definition Of Success

By the end of day 3:

- 2-3 sketches exist
- 1 strong collaborative demo is bounced
- bass recordings are saved and labeled
- you both want to come back to the best idea
